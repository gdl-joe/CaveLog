# CaveLog als App auf dem Handy installieren

CaveLog ist eine **Progressive Web App (PWA)** — das bedeutet: Sie verhält sich wie eine echte App, wird aber direkt aus dem Browser installiert. Kein App Store, kein Download.

Nach der Installation:
- Eigenes Icon auf dem Homescreen
- Öffnet sich ohne Browser-Adressleiste (wie eine richtige App)
- Funktioniert auch ohne stabiles Netz (offline-fähig)

---

## Android — mit Chrome

Chrome ist auf den meisten Android-Handys vorinstalliert und unterstützt PWA-Installation am besten.

### So geht's:

**1.** Öffne **Chrome** auf deinem Android-Handy

**2.** Rufe CaveLog auf:
```
https://deine-domain.de
```

**3.** Melde dich an

**4.** Nach kurzer Zeit erscheint automatisch ein Banner unten:
> **„CaveLog zur Startseite hinzufügen"**
>
> Tippe auf **„Hinzufügen"** → fertig!

---

**Falls das Banner nicht erscheint:**

**4b.** Tippe oben rechts auf die **drei Punkte** ⋮ (Menü)

**5.** Wähle **„App installieren"** oder **„Zum Startbildschirm hinzufügen"**

   *(Der genaue Name variiert je nach Android-Version)*

**6.** Im Bestätigungsdialog auf **„Installieren"** tippen

**7.** Das CaveLog-Icon erscheint auf deinem Homescreen ✓

---

### Tipp für Samsung-Geräte (Samsung Internet):

Falls du Samsung Internet nutzt:
1. Menü unten → **„Seite hinzufügen"**
2. **„Zum Startbildschirm"**

---

## iPhone / iPad — mit Safari

Auf Apple-Geräten muss **Safari** verwendet werden — andere Browser (Chrome, Firefox) unterstützen die PWA-Installation auf iOS leider nicht.

### So geht's:

**1.** Öffne **Safari** auf deinem iPhone/iPad

**2.** Rufe CaveLog auf:
```
https://deine-domain.de
```

**3.** Melde dich an

**4.** Tippe unten in der Mitte auf das **Teilen-Symbol** 

   *(Das Viereck mit dem Pfeil nach oben ↑)*

**5.** Scrolle im Menü nach unten bis **„Zum Home-Bildschirm"**

   *(Es kann sein, dass du etwas nach unten scrollen musst)*

**6.** Tippe darauf → Bestätige mit **„Hinzufügen"**

**7.** Das CaveLog-Icon erscheint auf deinem Home-Bildschirm ✓

---

### Wichtig für iPhone-Nutzer:

- Die App muss über **Safari** geöffnet werden (nicht Chrome oder Firefox)
- Beim ersten Öffnen erscheint kurz die Browser-Leiste, danach startet sie im Vollbild
- iOS 16.4+ hat verbesserte PWA-Unterstützung — bitte auf aktuellem iOS-Stand halten

---

## Desktop (Windows / Mac)

Auch am Computer kann CaveLog als App installiert werden:

**Chrome / Edge:**
1. Öffne CaveLog im Browser
2. Klicke in der Adressleiste rechts auf das **Install-Symbol** (⊕)
   *oder: Menü → „CaveLog installieren"*
3. Bestätige mit „Installieren"
4. CaveLog öffnet sich in einem eigenen Fenster ohne Browser-Adressleiste

---

## Die App deinstallieren

**Android:** Lang auf das Icon tippen → „Deinstallieren" oder „Entfernen"

**iPhone:** Lang auf das Icon tippen → „App entfernen" → „Vom Home-Bildschirm entfernen"

> Hinweis: Das Deinstallieren der App löscht keine Daten auf dem Server.

---

## Offline-Funktion

CaveLog speichert Teile der App lokal, sodass sie auch bei schlechtem Netz funktioniert:

- Die App-Oberfläche startet immer (auch ohne Netz)
- Zuletzt geladene Befahrungen sind weiterhin lesbar
- Neue Befahrungen werden zwischengespeichert und hochgeladen, sobald wieder Verbindung besteht

---

*Getestet mit: Android 12+/Chrome 120+, iOS 16.4+/Safari, Windows 11/Chrome 120+*
